# Anime to Anime Frontend
This is the frontend repository for the [Anime to Anime](https://sata.li/animetoanime/) web daily game.  
It is a static site built using React + TypeScript that recieves data from a [backend](https://github.com/satasatalight/animetoanime-backend).
